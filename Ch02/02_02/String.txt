### JavaScript Strings - Simple Notes  

#### 1. What is a String?  
A string is a sequence of characters used to store text.  

#### 2. Creating Strings  
You can create strings in three ways:  
```js
const str1 = "Hello";  // Double quotes
const str2 = 'World';  // Single quotes
const str3 = `Hello, ${str2}!`;  // Backticks (Template Literal)
```
ðŸ‘‰ Example Output: `"Hello, World!"`

#### 3. Accessing Characters  
- Using charAt(index) method:
  ```js
  "cat".charAt(1); // Output: "a"
  ```
- Using bracket notation:
  ```js
  "cat"[1]; // Output: "a"
  ```

#### 4. String Length  
```js
const text = "Hello";
console.log(text.length); // Output: 5
```

#### 5. String Methods  
| Method | Description | Example | Output |
|--------|-------------|---------|--------|
| `toUpperCase()` | Converts to uppercase | `"hello".toUpperCase()` | `"HELLO"` |
| `toLowerCase()` | Converts to lowercase | `"WORLD".toLowerCase()` | `"world"` |
| `concat()` | Joins two strings | `"Hello".concat(" World")` | `"Hello World"` |
| `indexOf()` | Finds position of text | `"Hello".indexOf("e")` | `1` |
| `substring()` | Extracts part of a string | `"Hello".substring(1,3)` | `"el"` |
| `replace()` | Replaces text | `"Hello".replace("H", "Y")` | `"Yello"` |
| `trim()` | Removes spaces | `"  Hello  ".trim()` | `"Hello"` |
| `split()` | Converts string to array | `"a,b,c".split(",")` | `["a", "b", "c"]` |

#### 6. Comparing Strings  
```js
const a = "apple";
const b = "banana";
console.log(a < b); // true (alphabetical order)
```

#### 7. String Objects vs. Primitives  
```js
const strPrimitive = "Hello";  // String primitive
const strObject = new String("Hello"); // String object

console.log(typeof strPrimitive); // "string"
console.log(typeof strObject);    // "object"
```
âš  Warning: Avoid using `new String()` because it creates an object instead of a simple string.
